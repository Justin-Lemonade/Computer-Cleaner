# Context Engineering Kit

## Playbook
- Start with a concise plan and file scope.
- Keep active notes in small, updateable checkpoints.
- Summarize decisions and unresolved blockers.
- End with an explicit handoff section.

## Guardrails
- Avoid repeating unchanged context.
- Record assumptions and verification commands.
