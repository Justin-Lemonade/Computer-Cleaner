# Claude Scientific Skills

## Workflow
1. Frame the scientific question and constraints.
2. State candidate hypotheses.
3. Define data and method requirements.
4. Evaluate evidence quality and limitations.
5. Produce reproducible conclusions with uncertainty notes.

## Guardrails
- Distinguish observed facts from inferences.
- Include uncertainty and confounders.
