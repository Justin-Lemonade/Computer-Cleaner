# Claude Mem

## Memory Schema
- DECISION: architecture and implementation choices.
- CONSTRAINT: hard limits and compatibility needs.
- PREFERENCE: user style and workflow preferences.
- TODO: unresolved action items.

## Usage
- Append short entries with ISO date stamps.
- Prefer updates over duplicate entries.
